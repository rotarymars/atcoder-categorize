まず、標準入力から n を入力します。
次に、ABC という文字列を出力します。
最後に、最初に入力した n を出力し、改行文字を出力します。

C++のコード例は以下です。

#include <iostream>
using namespace std;
int main() {
int n;
cin >> n; // n を入力
cout << "ABC"; // ABC を出力
cout << n; // n を出力
cout << endl; // 改行文字
return 0;
}
