A - Your First Judge 解説 by physics0523

プログラムに対する入力を文字列 \(S\) として受け取り、 \(S\) が Hello,World! と一致するかどうかを判定すればよいですが、ポイントとなる点が \(2\) 点あります。

・入力を受け取る/答えを出力する

あなたが用いるプログラミング言語によって、入力を \(1\) つの文字列として受け取ったり、答えを出力したりするときにとるべき方法は大きく異なります。また、一部のプログラミング言語にとってはこれは難しいことであるかもしれません。
標準入出力の方法を調べるためには、以下のような方法が考えられます。

  • その言語に関するドキュメントや記事などを参照する。
  • 過去に開催されたコンテストなどで、その言語で行われた提出を参照する。
・文字列同士を比較する

文字列同士の比較の方法も、プログラミング言語によって大きく異なります。文字列同士の比較演算子の定義が言語によって微妙に異なったり、そもそも文字列型が存在する言語と存在しない言語があったりもします。
しかし、大抵の言語には文字列を比較するためになんらかの方法が用意されているので、その使い方を理解しておくことが肝要です。

以下に様々な言語による実装例を示します。
一部の言語では copy ボタンで正常に copy&paste が行えません。ご注意ください。

Objective-C

#import <Foundation/Foundation.h>
#import <stdarg.h>

@interface StdIO : NSObject
+(NSString*)stringFromStdIn;
+(void)printf:(NSString*)format, ...;
@end

@implementation StdIO
+(NSString*)stringFromStdIn
{
    NSFileHandle *handle = [NSFileHandle fileHandleWithStandardInput];
    NSMutableString *result = [NSMutableString string];
    NSData *data = [handle availableData];
    while ([data length] > 0) {
        [result appendString:[[[NSString alloc] initWithData:data encoding:NSUTF8StringEncoding] autorelease]];
        data = [handle availableData];
    }
    NSUInteger length = [result length];
    if (length >= 1 && [result characterAtIndex:length-1] == '\n') {
        [result deleteCharactersInRange:NSMakeRange(length-1,1)];
    }
    return result;
}
+(void)printf:(NSString*)format, ...
{
    va_list arguments;
    va_start(arguments, format);
    NSString *string = [[[NSString alloc] initWithFormat:format arguments:arguments] autorelease];
    NSFileHandle *fileHandle = [NSFileHandle fileHandleWithStandardOutput];
    [fileHandle writeData:[string dataUsingEncoding:NSUTF8StringEncoding]];
    va_end(arguments);
}
@end

int main() {
    @autoreleasepool {
        NSString *S = [StdIO stringFromStdIn];
        if([S isEqualToString:@"Hello,World!"]){[StdIO printf:@"AC"];}
        else{[StdIO printf:@"WA"];}
    }
}
