「三⾓形の⾯積」=「底辺」×「⾼さ」÷ 2 です。よって、|AB| × |BC| ÷ 2 を出⼒すればよいです。
この問題では、|CA| の値は解答に影響しませんでした。今回は |CA| の値は最後に与えられる
ため、これを⼊⼒として受け取らなくても問題に正答できます。ですが、不都合が⽣じる場合もあ
るので (複数のテストケースがある, クエリ形式の問題である等) 慣れないうちは⼀応全て⼊⼒とし
て受け取る癖をつけておくことをおすすめします。
以下に C++ での解答例を⽰します。
#include <iostream >
using namespace std;
int main () {
int ab , bc , ca;
cin >> ab >> bc >> ca;
int area = ab * bc / 2;
cout << area << endl;
return 0;
}
