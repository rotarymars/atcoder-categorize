
Grouping 2

3 人以上のグループは最大で ⌊N/3⌋ 個作れます。
よって、 ⌊N/3⌋ を出力するとよいです。
# include < cstdio >
int N ;
int main ()
{
scanf ("% d " ,& N );
printf ("% d \ n " , N /3);
}
