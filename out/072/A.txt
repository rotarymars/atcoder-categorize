X > t なら X − t[g] 残っていて、X ≤ t なら 0[g] になっています。これは max(X − t, 0) と書くことも出
来ます。コードにすると次のようになります。
int main (){
int X , t ;
cin > >X > > t ;
if (X > t ) cout < <X -t < < endl ;
else cout < <0 < < endl ;
return 0;
}
